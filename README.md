# E-SUMMIT'26 Mobile App

A premium, futuristic mobile app UI for the E-SUMMIT'26 entrepreneurship event.

## Tech Stack
- React
- Vite
- React Router DOM
- Lucide React (Icons)
- Vanilla CSS (Variables & Utility Classes)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open the app in your browser (usually http://localhost:5173).

## Project Structure
- `src/components`: Reusable UI components (Button, Cards, Header, Nav).
- `src/pages`: Individual screen components.
- `src/index.css`: Global design tokens and styles.
